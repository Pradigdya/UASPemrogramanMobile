# Assets Images

Folder ini berisi gambar-gambar yang digunakan dalam aplikasi.

## File yang diperlukan:

1. **event1.jpg** - Gambar untuk event 1
2. **event2.jpg** - Gambar untuk event 2
3. **org_logo.png** - Logo organisasi (opsional)
4. **profile.png** - Foto profil pembuat aplikasi

## Catatan:

- Jika gambar tidak ditemukan, aplikasi akan menampilkan placeholder icon
- Pastikan ukuran gambar tidak terlalu besar untuk performa yang baik
- Format yang didukung: JPG, PNG

## Cara menambahkan gambar:

1. Copy gambar ke folder ini
2. Pastikan nama file sesuai dengan yang digunakan di kode
3. Jalankan `flutter pub get` jika diperlukan
4. Restart aplikasi
